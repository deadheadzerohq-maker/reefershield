# ReeferShield

ReeferShield is a SaaS platform for autonomous reefer monitoring and tamper‑proof compliance certificate generation.

This repository contains:

- `web/` – Next.js 14 App Router + Tailwind + shadcn‑style UI for the dashboard
- `backend/` – Python FastAPI backend for telematics webhooks, certificate generation, IPFS, and Polygon notarization
- `supabase_schema.sql` – base schema for Supabase Postgres

Follow the step‑by‑step guide inside `docs/SETUP.md`.
